# Learned Patterns - Education

Patterns specific to learning and skill acquisition.

## Learning Science Patterns

## Pattern: Spaced Repetition

**Trigger:** Forgetting what you learned  
**Principle:** Spaced reviews build long-term memory  
**Action:** Review material at increasing intervals (1, 3, 7, 14, 30 days)

## Pattern: Active Recall

**Trigger:** Passive reading without retention  
**Principle:** Retrieving info strengthens memory  
**Action:** Test yourself, don't just re-read

## Pattern: Interleaving

**Trigger:** Blocking same-type practice  
**Principle:** Mixing topics improves transfer  
**Action:** Practice different types together

## Pattern: Elaboration

**Trigger:** Surface-level understanding  
**Principle:** Explaining to yourself deepens learning  
**Action:** Ask "why" and "how" for each concept

## Study Patterns

## Pattern: Deep Work Blocks

**Trigger:** Fragmented, distracted study  
**Principle:** Uninterrupted time produces learning  
**Action:** 90-minute blocks, no interruptions

## Pattern: Sleep Integration

**Trigger:** Late-night cramming  
**Principle:** Sleep consolidates memory  
**Action:** Study, sleep, review the next morning

## Pattern: Teaching to Learn

**Trigger:** Understanding feels sufficient  
**Principle:** Teaching reveals gaps  
**Action:** Explain concepts as if to a beginner

## Resource Patterns

## Pattern: One Good Resource

**Trigger:** Jumping between resources  
**Principle:** Depth beats breadth  
**Action:** Master one resource before moving on

## Pattern: Project-Based Learning

**Trigger:** Passive consumption without application  
**Principle:** Making things cements learning  
**Action:** Build something with what you learn

## Pattern: Primary Sources

**Trigger:** Learning from secondary interpretations  
**Principle:** Original sources are most accurate  
**Action:** Go to the source when possible

## Motivation Patterns

## Pattern: Minimum Daily

**Trigger:** Inconsistent study habits  
**Principle:** Small daily progress beats occasional bursts  
**Action:** Study for minimum 15 minutes daily

## Pattern: Accountability Partner

**Trigger:** Lack of external motivation  
**Principle:** Social commitment increases follow-through  
**Action:** Find someone to check in with weekly

## Pattern: Progress Visibility

**Trigger:** Feeling stuck or stagnant  
**Principle:** Visible progress motivates  
**Action:** Track and review your learning journey

## Reference Information

### What Works
- Active recall
- Spaced repetition
- Elaborative interrogation
- Interleaved practice
- Concrete examples

### What Doesn't Work
- Rereading
- Highlighting
- Cramming
- Passive review
