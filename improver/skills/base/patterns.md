# Base Skill - Universal Patterns

## Pattern: Progressive Clarity

**Trigger:** User has vague goals  
**Principle:** Goals become clearer through questioning  
**Approach:** Start with "What does success look like?" then drill down

## Pattern: Effort Respect

**Trigger:** User has tried many things  
**Principle:** Failed attempts contain valuable information  
**Approach:** Ask what they learned, not just what failed

## Pattern: Constraint Inversion

**Trigger:** User sees constraints as blockers  
**Principle:** Constraints often reveal the real path  
**Approach:** "Given your constraints, what IS possible?"

## Pattern: Success Definition

**Trigger:** User hasn't defined success  
**Principle:** Without metrics, progress is invisible  
**Approach:** "How will you know you've succeeded?"

## Pattern: Resource Audit

**Trigger:** User feels stuck  
**Principle:** Unused resources are often hidden  
**Approach:** "What haven't you tried yet?"

## Pattern: Timeline Reality

**Trigger:** User has ambitious goals but no timeline  
**Principle:** Ambitious goals need milestone decomposition  
**Approach:** "What can you do in the first week?"

## Pattern: Stakeholder Map

**Trigger:** Problem involves others  
**Principle:** People have competing interests  
**Approach:** "Who benefits if you succeed? Who loses?"

## Pattern: Failure Analysis

**Trigger:** User mentions past failure  
**Principle:** Failure has lessons success hides  
**Approach:** "What would you do differently?"
