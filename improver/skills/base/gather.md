# Base Skill - Universal Question Templates

## Universal Gathering Questions

Use when no specific skill matches:

### Identity
- Who are you in this context?
- What is your current situation?
- What experience do you have?

### Goal
- What specific outcome do you want?
- How will you know you've achieved it?
- What is your timeline?

### Effort
- What have you already tried?
- What worked? What didn't?
- What barriers have you encountered?

### Resources
- What time can you dedicate?
- What budget do you have?
- What skills or tools do you have?
- Who else is involved?

### Context
- What external factors matter?
- Are there any non-negotiables?
- What would make this urgent vs. important?

## Follow-up Framework

```
If: User provides incomplete answer
Then: Ask clarifying question from same category

If: User mentions specific domain
Then: Load domain-specific skill

If: User has tried many things
Then: Focus on what was learned, not what failed

If: Timeline is unclear
Then: Ask "By when would you like to see results?"
```

## Question Adaptation Rules

1. **Match user language** - If they say "career", ask about career
2. **Build on answers** - Don't ask what they've already shared
3. **One priority at a time** - Focus on most impactful gap
4. **Close open loops** - Each question should feel like progress
