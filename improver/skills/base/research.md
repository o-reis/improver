# Base Skill - Research Framework

## Research Strategy

### Step 1: Problem Decomposition

Break the goal into components:
```
Goal: [User's stated goal]
├── What: The specific outcome
├── Who: Target audience or beneficiaries
├── How: Methods or approaches
├── Why: Underlying motivation
└── When: Timeline and milestones
```

### Step 2: Query Generation

For each component, generate queries:

| Component | Query Focus |
|-----------|-------------|
| What | Definitions, standards, examples |
| Who | Case studies, profiles, benchmarks |
| How | Methods, processes, tools |
| Why | Psychology, theory, principles |
| When | Timing, sequencing, milestones |

### Step 3: Source Types

| Source Type | Use For |
|-------------|---------|
| Academic | Principles, theories, research |
| Industry | Trends, benchmarks, practices |
| Personal | Case studies, anecdotes |
| Expert | Professional opinions |
| Community | Shared experiences |

### Step 4: Synthesis Approach

1. **Cluster** - Group related findings
2. **Prioritize** - Rank by relevance to user
3. **Contextualize** - Connect to user's situation
4. **Simplify** - Remove jargon, keep essence
5. **Actionify** - Convert to actionable insights

## Quality Criteria

- [ ] At least 3 diverse sources
- [ ] Recent data (prefer <2 years)
- [ ] Contradicting viewpoints noted
- [ ] Actionable takeaways identified
- [ ] User context addressed
