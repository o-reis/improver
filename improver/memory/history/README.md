# Problem History

This folder stores completed problem-solving sessions.

## Files

User-specific markdown files are stored here when `/improve` sessions complete.

## Privacy

This folder is gitignored by default to protect user privacy.

## Structure

```
history/
└── {YYYY-MM-DD}_{slug}.md
```

Example: `2026-04-09_rock-singer-success.md`

See `../STRUCTURE.md` for entry format documentation.
